# Wine
- 와인은 사용자가 다양한 와인을 검색하고 평가할 수 있는 와인 리뷰 플랫폼으로, 사용자가 직접 와인에 대한 정보를 작성하고, 다른 사용자들이 그 정보를 기반으로 리뷰를 남기는 서비스입니다. 
- DEMO : [Wine](https://wine-cdi.vercel.app/)

## 개발 기간
- 2024.08.30 ~ 2024.09.16 (2주)
